Introduction

Welcome to People API! You can use our API to access People API endpoints and get personal information of people from our database.

Authentication

This is a public API and does not require any API key for authentication.

People

GetAll

Returns an array of personal information of all people.

JSON:

[  
  {
    "Id": 1,
    "Name": "Mahesh",
    "Occupation": "IT",
    "Contact": 9876543210,
    "Address": "Paud Road, Pune",
    "Guid": null,
    "UDID": "0e3f9a93-7f29-4d2a-a371-d30c019c8667",
    "APKversion": "1.2"
  }  
]

Query Parameters

This endpoint does not require any query parameters.

GetPeople

Retrieves personal information of a specific person.

Request

Key Value
Id integer. The ID of the person whose information is to be retrieved.

JSON:

{
  "Id": 1  
}

Response

Key Value
Id long. The ID of this person.
Name string. The name of this person.
Occupation string. The occupation of this person.
Contact string. The contact details of this person, generally a mobile number.
Address string. The Address of this person.
Guid string. A globally unique identifier of the application installed on the machine of this person.
UDID string. A unique identifier of the device of this person.
APKversion string. The version of the software installed on the device of this person.

JSON:

  {
    "Id": 1,
    "Name": "Mahesh",
    "Occupation": "IT",
    "Contact": "9876543210",
    "Address": "Paud Road, Pune",
    "Guid": null,
    "UDID": "0e3f9a93-7f29-4d2a-a371-d30c019c8667",
    "APKversion": "1.2"
  }
